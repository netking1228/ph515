參考行程

第 1 天： 北環景點：
抵達澎湖 → 長髮公主 → 後寮天堂 → 通樑古榕 → 跨海大橋 → 鯨魚洞 → 小門地質館 → 大義宮 → 夢幻沙灘 → 二崁聚落 → 池西岩瀑 → 池東大菓葉玄武岩 → 三石壁 → 西嶼西臺 → 小池角雙曲橋 → 漁翁島燈塔 → 外垵漁港觀景台
第 2 天： 銀海浪漫之旅：
歧頭遊艇碼頭搭船出航～讓專業的為您導覽解說…
東海巡航：員貝後山三大景點介紹、澎澎灘會移動的小島、雞善嶼、錠鉤嶼
觀賞燕鷗：觀察燕鷗覓食俯衝入海
鳥嶼午餐：登島鳥嶼含午餐(海鮮粥+3道小菜)
澎澎灘水上活動： 1.透明獨木舟　 2.SUP立槳　 3.超大型水上溜滑梯　 4.高空跳水　 5.泰山式搖擺跳水　 6.香蕉船　 7.海戰車　 8.水上摩托車
第 3 天： 海洋牧場+市區：
海洋牧場 → 四眼井 → 中央老街 → 澎湖天后宮 →舊郵便局 → 順承門 → 篤行十村 → 張雨生故事館 → 潘安邦紀念館 → 觀音亭親水遊憩區 → 澎湖媽祖觀光文化園區
說明：
「 參考行程 」是假設選了"銀海浪漫之旅"和"海洋牧場"舉例，您可以依您的需求來作規劃。若上列的「 本島行程 」和「 離島行程 」不喜歡，也可以從我們上百個行程中選擇並多退少補。 上百種行程
